# Internal Complaint Management System

Ikilas is a **ticketing and complaint management system** developed during my internship at **Majlis Perbandaran Klang**.  
The system was built using **Laravel** to help staff report IT-related issues while enabling the IT department to efficiently track, manage, and resolve complaints.

---

## 🚀 Features
- Developed with **Laravel Framework (PHP)**
- Full **CRUD functionality** for complaint records
- **User authentication & role-based access control**
- **Approval workflow**: issues require verification from higher authorities (Head of Department, Director) before resolution
- Streamlined IT support by improving efficiency and transparency in issue handling

---

## 🛠️ Tech Stack
- **Backend:** PHP (Laravel)  
- **Database:** MySQL  
- **Frontend:** Blade Templates, HTML, CSS, JavaScript  
- **Version Control:** Git / GitHub

## 📖 How It Works
1. Staff submits a complaint through the system.  
2. Complaint is logged and tracked via **ticketing system**.  
3. IT Department addresses the issue.  
4. Resolution must be verified by higher authorities (e.g., **Ketua Jabatan**, **Pengarah**) before closure.  

---

## 🎯 Impact
- Improved complaint handling process within the organization  
- Ensured **accountability** with hierarchical approval  
- Reduced IT support response time
